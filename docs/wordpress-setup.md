# WordPress 首次设置清单

容器首次启动后，打开 `https://你的域名`，按 WordPress 安装向导创建唯一管理员账号。

## 启用本站功能

进入后台：

1. 在“外观 → 主题”启用 **哆啦D梦的口袋**。
2. 在“插件”启用 **哆啦D梦相册**。
3. 在“设置 → 阅读”中：
   - 设置首页显示为“静态页面”；
   - 创建“首页”页面并选择为首页；
   - 创建“文章”页面并选择为文章页。
4. 在“网站设置”中上传头像并填写站点名称与说明。首页头像会直接进入终端风格留言板。
5. 在“设置 → 固定链接”选择“文章名”，然后点击“保存更改”。这会生成文章详情页和 `/wall/` 留言板需要的重写规则。

## 管理留言板

访客可以从首页头像进入 `/wall/`，昵称允许留空，普通留言会立即公开，包含网址的留言会进入待审核。

后台“留言板”可以公开、隐藏、置顶、回复或删除留言，也可以封禁恶意来源。导出的内容迁移包会包含留言、回复和点赞数。

## 设置首页大图

首页会自动轮播最近相册的封面和照片，最多显示 6 张。通过“外观 → 自定义 → 首页精选照片”选择的图片会固定为轮播第一张。

## 创建第一本相册

1. 打开“照片 → 新建相册”。
2. 填写相册名称。
3. 点击“批量上传或选择照片”，可一次上传多张照片，也可从照片库选择已有照片。
4. 拖拽缩略图调整顺序；点击照片下方的“设为封面”即可更换封面。
5. 直接发布。年份已经默认填好，地点和简短说明都可以留空或以后再补。

“照片 → 照片库”用于查看网站里的全部照片。日常上传建议从“新建相册”进入，这样照片会直接归入对应相册。

WordPress 会为上传图片自动生成多种尺寸。上传到网站的是公开展示版本；相机原图和 RAW 请继续保存在本地与离线备份中。

## 从 Markdown 导入文章

1. 打开后台“文章 → 导入 Markdown”。
2. 选择一个 UTF-8 编码的 `.md` 或 `.markdown` 文件，最大 2 MB。
3. 点击“导入并打开编辑器”。网站会创建草稿，不会直接发布。
4. 在编辑器里检查正文，并按需设置标签、封面和发布时间。

文件首个非空行如果是 `# 一级标题`，会被用作文章标题并从正文中移除；否则使用文件名作为标题。段落、各级标题、列表、引用、链接、图片、表格和代码块会转换成 WordPress 可编辑内容。

## 记录动画与关联文章

1. 打开后台“异世界 → 记录动画”，手动填写名称、海报、年份、集数、个人评分与记录。
2. 评分使用十分制，支持 `0.5` 分步进；发布后总榜会自动按评分从高到低排列，未评分项目排在末尾。
3. 编辑普通文章时，在右侧“关联动画”中搜索并勾选一部或多部动画。
4. 动画详情会自动列出相关文章，文章详情也会显示所关联的动画。

“备份迁移”生成的 ZIP 会同时包含动画资料、海报图片和文章关联关系。
