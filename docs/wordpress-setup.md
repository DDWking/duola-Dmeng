# WordPress 首次设置清单

容器首次启动后，打开 `https://你的域名`，按 WordPress 安装向导创建唯一管理员账号。

## 启用本站功能

进入后台：

1. 在“外观 → 主题”启用 **哆啦D梦的口袋**。
2. 在“插件”启用 **哆啦D梦相册**。
3. 在“设置 → 阅读”中：
   - 设置首页显示为“静态页面”；
   - 创建“首页”页面并选择为首页；
   - 创建“文章”页面并选择为文章页。
4. 创建一个标题为“归档”、固定链接为 `archive` 的页面。主题会自动使用归档模板。
5. 创建一个标题为“关于”、固定链接为 `about` 的页面，只填写一段简短介绍文字。
6. 在“外观 → 菜单”创建主导航，顺序为：首页、摄影、文章、归档、关于；将其分配到“主导航”。

## 设置首页大图

第一版通过“外观 → 自定义 → 首页精选照片”选择首页大图。没有设置图片时，首页会使用蓝色渐变背景，其他功能不受影响。

## 创建第一本相册

1. 打开“相册 → 新建相册”。
2. 填写标题和年份。
3. 点击“批量添加照片”，可一次选择多张照片上传或从媒体库选择。
4. 拖拽缩略图调整顺序；可选择一张作为相册封面。
5. 直接发布。地点、说明、单张照片说明均可留空。

WordPress 会为上传图片自动生成多种尺寸。上传到网站的是公开展示版本；相机原图和 RAW 请继续保存在本地与离线备份中。
