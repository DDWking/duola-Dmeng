# 瓦力波

使用 Godot 4.7 制作的横版单人 1v1 动作排球游戏，专注电脑 Web 与桌面键盘体验。

本目录是网站仓库中的唯一游戏源码。`../web/runtime` 与 `../server/project` 由仓库根目录的 `scripts/sync-volleyball.ps1` 生成，不应手动修改。

## 游戏内容

- 单人对战固定普通 AI，无需额外选择难度
- 房间码互联网 1v1：游客昵称、权威服务器、30 秒断线重连
- 个人网站访客积分榜：只记录单人完整比赛，每位访客展示最高分
- 两名定位鲜明的原创角色：高大强攻的炎哥、矮小灵活的炎弟
- 固定三触球流程：垫球、二传、起跳扣球；失误时仍可保命回球
- 移动、跳跃、拦网和飞身救球
- 快速下落、方向技能、挥空硬直和网口对抗球等进阶动作机制
- 拦网拥有独立冷却、拦空硬直和连续使用疲劳
- 每方最多触球三次
- 11 分制，领先 2 分获胜，三局两胜
- 暂停、比赛结算与本地胜负战绩
- 中文按键设置界面，支持八项操作重绑定、重复按键自动交换与恢复默认

## 操作

电脑键盘：

- `A` / `D`：移动
- `S`：空中快速下落
- `W`：跳跃
- `J`：双脚着地时完成第一触球垫球和第二触球二传；第三触球时地面保命回球、空中吊球
- `K`：只在第三触球且位于空中时执行向下重扣；地面按下无动作
- `A` / `D` + `K`：第三次触球执行斜线扣球
- `L`：地面飞身救球；空中无动作
- `U`：拦网

默认按键可在选人界面的“按键设置”中修改，设置会保存到本地。

单人比赛会按胜负、局数、扣球、救球、拦网、完美触球和最高连击结算积分。昵称可以留空，成绩会以“匿名球员”登记；联机对战不计入榜单。

一传和二传的判定质量会连续影响轨迹：timing 越好，球越高且越靠近人物；timing 较差时，球会沿当前人球角度低飞并远离人物。前场扣球 timing 越好，向下扣球的速度越快。

距离球网超过 150 像素时，扣球会自动改为带弧线的后场包球：先安全越过网带，再向对方场内下坠；timing 越好，包球速度越快。前场仍使用直接向下的重扣轨迹。

电脑面对后场包球时以接球为主，每次来球只有 15% 概率选择拦网；这次防守选择会锁定到球权结束，不会逐帧重复抽取。前场扣球仍沿用原有拦网判断。

## 启动

双击 `run_game.bat`，或使用 `D:\Godot\Godot_v4.7.1-stable_win64.exe` 打开本目录。

## Web 导出

工程已配置 Web 导出预设，输出到 `build/web/index.html`。本地试玩：

`python -m http.server 8765 --bind 127.0.0.1 --directory build/web`

只安装 Godot Web 模板可运行：`python tools/install_web_templates.py`。

## 联机模式

双击 `run_multiplayer_server.bat` 启动本机权威服务器，默认监听 `9001`。游戏联机大厅默认连接 `ws://127.0.0.1:9001`，支持创建六位房间码、加入房间和恢复掉线房间。

腾讯云与个人网站的 `wss://` 配置见 `docs/multiplayer_deployment.md`。
